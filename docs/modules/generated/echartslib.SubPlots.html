The layout orientation of toolbox’s icon, by default ‘horizontal’

Distance between toolbox component and the right side of the container., by default ‘1%’

Distance between toolbox component and the top side of the container., by default ‘auto’

The tools included in toolbox. Options:

set_tooltip(show: bool = True, trigger: str = 'item', triggerOn: str = 'mousemove|click', type: str = 'shadow', formatter: Optional[str] = None)¶

Set tooltip config

Parameters¶

showbool, optional

Whether to show the tooltip, by default True

triggerstr, optional

Type of triggering. Options:

‘item’:Triggered by data item, which is mainly used for charts that don’t have a category axis like scatter charts or pie charts. ‘axis’:Triggered by axes, which is mainly used for charts that have category axes, like bar charts or line charts. ‘none’Trigger nothing.

, by default ‘item’

triggerOnstr, optional

Conditions to trigger tooltip. Options:

‘mousemove’: Trigger when mouse moves. ‘click’: Trigger when mouse clicks. ‘mousemove|click’: Trigger when mouse clicks and moves. ‘none’: Do not triggered by ‘mousemove’ and ‘click’.

, by default ‘mousemove|click’

typestr, optional

Indicator type. Options:

‘line’: line indicator. ‘shadow’: shadow crosshair indicator. ‘none’: no indicator displayed. ‘cross’: crosshair indicator, which is actually the shortcut of enable two axisPointers of two orthometric axes.

, by default ‘shadow’

formatterstr, optional

The content formatter of tooltip’s floating layer which supports string template and callback function. The template variables are {a}, {b}, {c}, {d} and {e}, which stands for series name, data name and data value and ect. When trigger is set to be ‘axis’, there may be data from multiple series. In this time, series index can be refered as {a0}, {a1}, or {a2}. {a}, {b}, {c}, {d} have different meanings for different series types: Line (area) charts, bar (column) charts, K charts: {a} for series name, {b} for category name, {c} for data value, {d} for none; Scatter (bubble) charts: {a} for series name, {b} for data name, {c} for data value, {d} for none; Map: {a} for series name, {b} for area name, {c} for merging data, {d} for none; Pie charts, gauge charts, funnel charts: {a} for series name, {b} for data item name, {c} for data value, {d} for percentage. Example: formatter: ‘{b0}: {c0}<br />{b1}: {c1}’ , by default None
