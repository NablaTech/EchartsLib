echartslib.histplot¶

echartslib.histplot(x, y=None, label=None, bins: int = 10, stat: str = 'count', multiple: str = 'layer')¶

Draw histogram plot

Parameters¶

xnp.darray, List

Data element 1

ynp.darray, List, optional

Data element 2, by default None

labelnp.darray, List, optional

The label of data class, by default None

binsint, optional

Histogram bins, by default 10

statstr, optional
Aggregate statistic to compute in each bin, by default ‘count’

  • count: show the number of observations in each bin

  • percent: normalize such that bar heights sum to 100

  • density: normalize such that the total area of the histogram equals 1

multiplestr, optional
Way of display for multiple bar plot, by default ‘layer’

  • layer: Put bar overlapping each other

  • stack: Put bar stack on each other

Returns¶

Figure

The figure object
